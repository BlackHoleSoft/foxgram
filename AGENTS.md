# Проект Foxgram

Мессенджер с клиент-серверной архитектурой и сквозным шифрованием.

## Процесс разработки

В проекте используется подход Spec driven development

Все спеки лежат в specs/[feature_name]/spec.md

При реализации фичи первым делом необходимо написать тесты (изначально все не проходят FAIL)
Все тесты описаны в спецификации, не нужно придумывать свои тесты, следуем строго спецификации.

После этого пишется код, дополняются тесты.

В итоге каждый тест должен проходить PASS
